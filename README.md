## Description

This is just for fun.
I've tested prisma ORM and speed of inserting simple records to PostgreSQL and MongoDB(replica set)
See Swagger OpenAPI on localhost

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
